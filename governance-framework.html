我們不直接替市長打分數，
而是拆解「治理能力」本身。

政策成果回答「做了什麼、結果如何」；治理能力回答「有沒有能力把事情做成」。這頁公開12項能力的定義、六任市長的完整1–5分評分、台北當代最需要強化的能力排序，以及——這次更新的重點——用GitHub Issues公開接受評分異議。

05

對評分有異議？

這頁的評分是研究團隊的推論，不是唯一正確答案。如果你認為某個分數不合理、有更好的證據，我們用 GitHub Issues 當公開的異議紀錄——任何人都可以看到誰提出了什麼質疑、我們怎麼回應，而不是私下修改分數就算了。

怎麼提出異議：到 本專案的 GitHub Issues 頁面 開一個新 issue，標題請包含市長姓名與能力項目（例如「柯文哲—國際合作評分」），內容附上你認為更準確的證據來源。我們會公開回覆，同意調整的話會附上修改紀錄。
方法論提醒：本頁的能力定義為研究團隊原創架構，優先排序是依現有趨勢資料的推論，並非學術文獻、專家共識調查或民調結果。我們刻意選擇公開推論過程，而不是只給結論，因為這個網站的核心主張就是「用證據不用立場」——如果連自己的方法論都不敢公開檢視，這個主張就站不住腳。完整資料來源見配套文件。
