# 台北市長政策回顧研究（1990–2026）

> 本文件為「跳開藍綠：台北市需要什麼樣的市長」專案的第一階段成果——資料整理與初步分析。目的是盡可能中立地整理歷任市長在**交通、商業/經濟發展、教育、社會福利、公共衛生/環境**五大領域的重大政策，並比對「當時的聲望／民調」與「事後的成效評價」，找出可能的落差與規律。之後可依此製作互動式網站（時間軸、雷達圖、政績-滿意度對照表）。
>
> **範圍說明**：應使用者要求，本版加入了1994年首次市長直選以前、最後一任「官派」市長黃大洲（1990–1994）。他雖非民選，但任內奠定的捷運、大安森林公園等基礎設施，是理解後續三十多年台北市發展的重要背景，也提供一個有趣的對照組——「不靠民意支持、卻做了大量長期建設」的治理模式，可與後續民選市長對照討論。

## 0. 研究框架說明

每位市長的紀錄採用同一個表格結構，方便日後轉成資料庫欄位：

| 欄位 | 說明 |
|---|---|
| 任期 | 起訖年份 |
| 當選背景 | 得票率、對手、政黨版圖 |
| 交通 | 捷運、公車、YouBike、停車、道路 |
| 商業/經濟 | 招商、都更、財政、國際排名 |
| 教育 | 政策與資源投入 |
| 社福/居住 | 社會住宅、津貼、長照、托育 |
| 公共衛生/環境 | 防災、環保、公共衛生事件應對 |
| **都市更新** | 都更案件量、公辦都更、危老重建、大型都市更新計畫 |
| **市民幸福感（人口移出入）** | 台北市對內政部人口統計的淨遷徙趨勢，作為「市民是否願意留在台北」的間接指標 |
| **國際參與** | 國際賽事主辦、城市外交/姊妹市、全球城市競爭力評比排名 |
| 當時聲望 | 任內民調高低點與變化 |
| 事後評價/爭議 | 政績歸屬爭議、後續是否被否定或延續 |

**重要提醒（資料侷限性）**：
- 台灣的民調機構、統計口徑（如社宅完工戶數怎麼算）常因政治立場而有爭議，本文件盡量標註「爭議點」而非單一定論。
- 早期市長（陳水扁、馬英九）之後皆曾任總統，部分「政績」在網路上會與其總統任內的建設混淆（例如雪隧、高鐵常被誤植為市長政績），本文件已盡量做區分。
- 「政績接棒」現象很普遍：一任市長規劃、下一任完工剪綵（如貓空纜車、大巨蛋、世大運場館），評價時應留意。

## 1. 黃大洲（1990–1994，官派，國民黨）

**背景說明**：黃大洲是台北市改制直轄市後**最後一任官派市長**（非民選）。1990年5月以市府秘書長身份代理市長，同年10月經市議會同意正式出任，1993年獲市議會同意續任。1994年代表國民黨參選首屆民選市長，因與新黨趙少康分票，落居第三名，敗於陳水扁。

由於是官派、不需直接面對選票壓力，他的施政風格與後續六位民選市長形成明顯對照，適合作為研究的「基準線」。

| 領域 | 重大政策/事件 |
|---|---|
| 交通 | 啟動台北捷運「六線齊發」同步施工（此前捷運議題已研議規劃逾10年，僅北投機廠於1988年動工，實際大規模開挖是在黃大洲任內）；捷運淡水線興建 |
| 商業/經濟 | 拆除8棟中華商場（西門町一帶知名老舊商場，拆遷抗爭規模空前）；台北市政大樓（信義區）啟用 |
| 教育/文化 | 台北市立圖書館新總館啟用 |
| 社福/都市發展 | 拆除七號公園違章建築區（原眷村與違建聚集地，拆遷戶逾2,600戶，過程中黃大洲本人曾遭恐嚇威脅），改建為**大安森林公園** |
| 公共衛生/環境 | 基隆河截彎取直工程（大直、內湖河段），為後續內湖科學園區、河濱公園開發奠定基礎；內湖垃圾焚化廠啟用；山豬窟垃圾掩埋場啟用 |

**都市更新／人口移出入／國際參與（新增維度）**：
- 都市更新：拆除中華商場、拆遷七號公園違建改建大安森林公園，屬於大規模「拆遷重建」，但當時尚無「都市更新」的正式法制概念（《都市更新條例》於2000年後才立法），故此處評分視為都更概念的前身。
- 人口移出入：1990年代台北市仍處於人口淨移入階段，雙北仍是全台人口磁吸中心，市民「留在台北」的意願整體偏高。
- 國際參與：任內國際型活動較少，以捷運等內部基礎建設為主。

**當時聲望**：偏低。黃大洲不擅言詞、不懂媒體公關，1994年選舉辯論會上被新黨趙少康公開嘲諷（「弄個大安森林公園，一下雨就變黃大洲」），選舉時多項建設（尤其捷運）在民間評價兩極，甚至有候選人（包括陳水扁）曾主張「當選後拆掉捷運」。最終選舉落居第三名。

**事後評價/爭議**：
- 這是本研究中「當時聲望」與「事後歷史評價」落差**最大**的案例：近年（尤其2020年代）多篇媒體回顧文章（《關鍵評論網》、《中國時報》等）重新評價黃大洲，稱其捷運路網、大安森林公園、中華商場拆除、基隆河截彎取直「四大政績」是「奠定台北三十年發展基礎」的關鍵建設，甚至有評論稱其貢獻「超扁勝馬郝」。
- 這類「當年挨罵、多年後被追認政績」的現象，也提醒我們評價一位市長時，**短期民調與長期歷史評價可能完全相反**，這對網站設計「聲望 vs. 成效」對照機制是重要的方法論提醒：許多長期基礎工程（截彎取直、大型公園、路網骨幹）的效益要10–20年才能完全顯現，用任內民調來評價這類政策本身就有結構性侵限。
- 也因為是官派而非民選，他不需要靠短期有感政策來討好選民，這使他得以推動高爭議、長回收期的建設——這也是「官派 vs. 民選」治理模式對照的核心觀察點，後續網站可以此設計專題（例如：「如果台北市長不用選舉，城市會不會不一樣？」）。
- 需注意：捷運路網最早的規劃研議可追溯至更早的張豐緒、林洋港、吳伯雄等歷任市長，黃大洲是「大規模動工執行者」而非「發起規劃者」，這也是政績歸屬需要精確描述的一例。

## 2. 陳水扁（1994–1998，民進黨）

**當選背景**：台北市改制直轄市後首任民選市長，1994年三腳督（黃大洲、趙少康、陳水扁）中以43.67%當選。1998年競選連任以45.91%敗於馬英九。

| 領域 | 重大政策/事件 |
|---|---|
| 交通 | 接手前任留下的捷運木柵線亂局，任內完成首條捷運通車；鐵腕整頓交通違規、拆違建 |
| 商業/經濟 | 促成台北101（BOT模式首例，實際完工在後續市長任內）；西門町徒步區規劃 |
| 教育 | 相對少被提及，非其主打領域 |
| 社福 | 社會局改革（陳菊任社會局長），推動多項社福制度化 |
| 公共衛生/治安 | 「一清專案」式掃黃掃黑、電子遊樂場稽查、公務員風氣改革 |
| 財政 | 4年減少債務約34億元 |

**都市更新／人口移出入／國際參與（新增維度）**：
- 都市更新：非其主打領域，西門町徒步區規劃屬商圈活化，非正式都更案。
- 人口移出入：延續1990年代人口淨移入趨勢，但增速已開始緩降。
- 國際參與：以內政治理（掃黃掃黑）為主軸，國際能見度相對較低。

**當時聲望**：任內滿意度一度高達近八成，是四位直轄市長中民調表現最突出的一位。

**事後評價/爭議**：
- 連任卻落敗，形成「滿意度高、得票率卻不夠」的知名落差案例，常被拿來討論「政績」與「選票」不一定成正比。
- 2020年代網路民調（如中時2018年調查）中，仍有多數受訪者認為他是史上最佳台北市長之一，但**評價存在明顯世代差異**：25歲以上滿意度較高，24歲以下（未經歷其市政時期）認同度較低——這是評價「政績」時必須留意的樣本偏誤。
- 部分建設（如垃圾隨袋徵收、環東大道）被視為「扁馬各半」，政績歸屬本身有爭議。
- 需與其後續總統任內政績（雪隧、高鐵動工於扁之前）做區隔，避免張冠李戴。

## 3. 馬英九（1998–2006，國民黨）

**當選背景**：1998年以51.13%擊敗尋求連任的陳水扁；2002年連任得票率64.11%，為歷來最高。

| 領域 | 重大政策/事件 |
|---|---|
| 交通 | 捷運路網大幅擴張、河濱自行車道系統、藍色公路（成效有限） |
| 商業/經濟 | 悠遊卡全面推廣；但債務8年間增加約1,332億元，是爭議焦點之一 |
| 教育 | 未特別突出，延續一般教育投入 |
| 社福 | 廣設平價健身中心、體育處成立 |
| 公共衛生/環境 | 垃圾費隨袋徵收（2001年獲亞洲廢棄物管理傑出獎）、污水下水道接管率40%→80% |
| 災害應變 | 經歷921大地震（上任第2年）、納莉風災、SARS，建立國際搜救隊與防災機制 |

**都市更新／人口移出入／國際參與（新增維度）**：
- 都市更新：競選承諾「反轉軸線、改造舊市區」，但實際成果不佳——建成圓環改造失敗、龍山寺地下街規劃不當閒置至今，是都更成果最弱的任期之一。
- 人口移出入：人口淨移入趨勢持續但增速趨緩，尚未轉為淨遷出。
- 國際參與：無重大國際賽事在其市長任內舉辦，但國際排名與媒體形象整體正向。

**當時聲望**：多次民調滿意度逾70–79%，是形象與民調雙高的少數案例。

**事後評價/爭議**：
- 承諾跳票項目明確：內湖捷運延宕、台北大巨蛋（原稱台北巨蛋）未如期完成、圓環改造失敗、龍山寺地下街規劃不當閒置。
- 弊案數量在其任內被在野立委列舉高達70餘件（如瀝青工程弊案），但多數未直接波及馬本人聲望。
- 財政負債大幅增加，是後來檢討「政績亮麗 vs. 財政永續」時常被提出的對照案例。
- 部分建設争议同陳水扁時期一樣，存在「規劃在前任、完工在馬任內」歸屬模糊的情況。

## 4. 郝龍斌（2006–2014，國民黨）

**當選背景**：2006年、2010年兩度當選，第二任大勝對手蘇貞昌17萬票。

| 領域 | 重大政策/事件 |
|---|---|
| 交通 | YouBike上線（前30分免費，滿意度高達93%）；捷運文湖線初期故障頻繁、貓空纜車因塔柱掏空長期停駛，是任內知名負面事件 |
| 商業/經濟 | 花博（2010，896萬參觀人次，430億經濟效益）；爭取2017世大運主辦權（實際舉辦在下任柯文哲任內） |
| 教育 | 相對少突出政策 |
| 社福/居住 | 開始推動公營住宅（社會住宅前身）；「助妳好孕」生育政策；1999市民熱線（滿意度最高的單項施政） |
| 公共衛生/國際交流 | 2009台北聽障奧運（獲國際組織高度評價）；創辦「雙城論壇」（台北-上海） |
| 環境爭議 | 為辦花博、大巨蛋、世大運選手村合計砍除超過3,000株樹木，引發環保爭議，與同期舉辦「國際樹木保育研討會」形成諷刺對比 |

**都市更新／人口移出入／國際參與（新增維度）**：
- 都市更新：開始推動公營住宅與都市更新制度化（如延宕多年的斯文里三期案於其任內啟動研議，但實際動工要等到柯文哲任內），屬於「起步但未見成果」階段。
- 人口移出入：淨遷入持續但明顯趨緩，任期末（2014年）台北市首次出現淨遷出（約0.5萬人），是「由正轉負」的轉折任期。
- 國際參與：**國際參與的高峰期**——2009台北聽障奧運、2010台北花博（896萬參觀人次）、創辦台北-上海雙城論壇、爭取到2017世大運主辦權（實際舉辦於下任），密集的國際型活動帶動城市能見度。

**當時聲望**：滿意度長期「不上不下」——國際獎項與評比屢創佳績，但台北市民主觀滿意度始終追不上外界掌聲（其本人亦曾公開自嘲此落差）。

**事後評價/爭議**：
- 網路輿論普遍評價「溫和、無明顯過失，但也缺乏強烈記憶點」（PTT討論常見用語：「馬規郝隨」）。
- YouBike、花博普遍被認為是實質政績；貓纜、文湖線故障則是明顯負面案例。
- 世大運主辦權是他爭取、由下任柯文哲完成舉辦，屬於典型「跨任期政績接棒」。

## 5. 柯文哲（2014–2022，無黨籍→台灣民眾黨）

**當選背景**：首位台北直轄市無黨籍市長，2014年以「白色力量」訴求當選，2018年連任。

| 領域 | 重大政策/事件 |
|---|---|
| 交通 | 停車格全面收費（爭議大）；雙北1280月票（累計售出逾1,300萬張）；取消YouBike前30分免費（使用量立即下滑20%）；捷運新線動工進度緩慢，是任內較弱項目 |
| 商業/經濟 | 西區門戶計畫（拆忠孝橋引道、北門重現）；財政紀律：8年還債570億元；連續4年獲招商卓越獎 |
| 教育 | 實驗教育擴大、數位實驗高中設立 |
| 社福/居住 | 社會住宅是其招牌政策，任內完工約1.1–1.2萬戶（若併計前朝動工部分約2萬戶，惟精確歸屬各方統計不一，詳見下方查核說明）；囤房稅全國首例 |
| 公共衛生/環境 | 「海綿城市」政策、抽水站全面智慧監控、共融式遊戲場（帶動其他縣市跟進） |
| 治理方式 | 開放政府、參與式預算（477件提案中388件結案，結案率81%）、資料開放平台 |

**都市更新／人口移出入／國際參與（新增維度）**：
- 都市更新：西區門戶計畫（拆忠孝橋引道、北門重現）、東西區門戶計畫，並完成延宕20多年的斯文里三期公辦都更（2017年接手、2022年落成），是「政府自行實施」都更的指標案例。
- 人口移出入：**任內出現關鍵轉折**——台北市自2014年起淨遷出，且逐年擴大：2016–2018年約每年淨遷出2萬人，2020年擴大至4.4萬人，2021年（卸任前一年）高達7.6萬人，創歷年新高。高房價與鄰近縣市交通改善是主要成因。
- 國際參與：舉辦2017世大運（實際主辦）、北流/北藝等國際級文化場館開幕、與布拉格締結姊妹市、當選「全球百大數位首長」，全球城市運轉指數亞洲排名第5。

**當時聲望**：第一任期滿意度高企，但**第二任期出現明顯下滑**，2020年天下雜誌調查中在六都首長滿意度排名墊底；主要原因被學者歸因於「市政與黨政分際不清」（成立台灣民眾黨、籌劃參選總統，被認為分心市政）。

**事後評價/爭議**：
- 社會住宅數字是台灣事實查核中心多次查核的焦點：柯文哲任內實際新動工並完工的社宅約5,000餘戶，若將前朝（郝龍斌）動工部分計入才達2萬戶量級——顯示「政績量化」本身容易被各方各取所需地詮釋。
- 大巨蛋爭議貫穿其兩任任期（停工、復工、防火避難審查延宕5年以上），卸任後仍未完工，成為後續蔣萬安任內完工的「三代政績接棒」典型案例。
- 卸任後因京華城容積案、政治獻金申報等問題遭起訴並被判刑，這是評價其市政遺產時無法迴避、但也需與其市政表現本身分開看待的部分。
- 房價：2016–2022年台北市房價指數漲幅是六都中最小的，這點常被其支持者引用為政績。

## 6. 蔣萬安（2022–現任，國民黨）

**當選背景**：2022年當選，2026年11月尋求連任，對手為民進黨徵召的沈伯洋（不分區立委）。

| 領域 | 重大政策/事件（至2026年中） |
|---|---|
| 交通 | 捷運「六線齊發」同步施工；持續推動都市更新（「都更八箭」，報核案件數為過去8年2.3倍） |
| 商業/經濟 | 促成輝達（NVIDIA）台灣總部落腳北士科（北士科T17/T18用地），為其任內最常被提及的招商政績；北士科用電議題（變電所地下化）也成為選戰焦點之一 |
| 教育/福利 | 營養午餐全面免費政策，被其他縣市（含綠營執政縣市）跟進效法，是少見「跨黨參考」的地方政策外溢案例 |
| 建設完工 | 台北大巨蛋正式啟用（歷經馬英九規劃、柯文哲延宕，蔣萬安任內完工開幕），是三代市長政績接棒的代表案例 |
| 危機處理 | 2026年上半年鼠患（含漢他病毒死亡案例）成為施政滿意度的短期扣分項目：民調顯示認為鼠患嚴重者近半數（49.2%），但認為蔣有能力解決者僅約39.4%，顯示「危機處理信心」與「整體滿意度」出現落差 |

**都市更新／人口移出入／國際參與（新增維度）**：
- 都市更新：主打「都更八箭」，報核案件數為過去8年的2.3倍，也高於其他五都加總的1.4倍，是六任市長中都更政策力度最強的一位（但實際完工成果尚待時間驗證）。
- 人口移出入：淨遷出情況延續且反覆——2023年因海外除籍者「復籍潮」曾短暫回流3.5萬人，但2024–2025年再度惡化，2025年淨遷出擴大至4.4萬人；市府已成立「人口對策委員會」應對。2016–2025十年間台北市累計淨遷出27.1萬人（約等於一個大安區人口），是六任市長中人口流失最嚴峻的階段。
- 國際參與：促成輝達（NVIDIA）台灣總部落腳北士科，屬於「經濟投資型」國際參與（相較郝龍斌、柯文哲的「賽事/文化型」國際參與是不同路徑）；傳統城市外交/國際賽事主辦相對較少。

**當時聲望（截至2026年中，仍在任）**：
- 上任初期（2023年1月）滿意度僅39%，一週年提升至65%。
- 2026年初多次調查滿意度在58–69%區間波動（TVBS、市府研考會、遠見雜誌調查結果略有差異，但方向一致：長期偏高、短期因鼠患議題小幅下滑）。
- 2026年九合一選舉對戰民調：蔣萬安支持度52–58%，大幅領先沈伯洋（29–30%），年輕族群（20–29歲）支持度更高達66%，顯示其民調基礎相對穩固。

**事後評價/爭議（進行中，尚未定論）**：
- 因任期未滿，「事後成效評價」目前多為即時性民調，尚缺乏跨任期的長期回顧視角，這也是30年回顧中唯一一位「現任、可比對資料最不完整」的市長，網站上線後可持續追蹤更新。
- 都更成果與大巨蛋完工屬於較實質可驗證的建設型政績；輝達進駐則屬於「招商型」政績，其長期經濟外溢效益仍待觀察。

## 7. 跨任期規律觀察（初步）

整理六任市長後，可以看到幾個重複出現的模式，這些會是網站「未來需要什麼樣市長」討論的重要素材：

0. **短期民調與長期歷史評價可能完全相反**：黃大洲是最極端案例——任內聲望低、選舉落敗，卻在20–30年後被廣泛追認為「奠定台北基礎」的市長。這提醒我們，任何只看「當時民調」的評價機制，都可能低估長回收期的基礎建設。
1. **「政績接棒」是常態，不是例外**：捷運路網、大巨蛋、世大運場館、貓纜復駛，幾乎每一項大型建設都跨越至少兩任市長。單一市長「政績歸屬」的爭論，往往簡化了城市治理本質上是接力賽的事實。
2. **「有感政績」與「基礎工程」的滿意度落差**：YouBike、營養午餐免費、悠遊卡這類直接觸及日常生活的政策，滿意度飆升速度遠快於污水下水道接管率、財政紀律這類長期才看得到效果的基礎工程——但後者對城市長期競爭力可能更關鍵。
3. **民調高峰與連任結果不一定相關**：陳水扁高滿意度卻連任失敗（1998），顯示地方選舉受政黨版圖、對手強弱、單一議題衝擊（如319槍擊案效應之於2004年全國選情）影響，遠比單純施政滿意度複雜。
4. **黨政分際模糊會侵蝕市政滿意度**：柯文哲第二任期滿意度下滑、蔣萬安目前也面臨「是否過度介入全國政治議題」的類似檢驗，顯示台北市長作為「潛在總統候選人」的身分，本身就是雙面刃。
5. **環境/永續政策長期被邊緣化**：五任市長中，環境政策多附屬於其他大型建設（花博、世大運附帶大量砍樹），較少看到以環境永續為主軸、且民調上獲得同等重視的市政週期。
6. **爭議與政績常常並存於同一政策**（大巨蛋、社宅數字、YouBike收費），這提示未來的評價機制應避免「單一指標打分」，而需要多維度並陳。
7. **人口淨遷徙是最誠實、也最容易被忽略的「市民用腳投票」指標**：台北市大約從郝龍斌任期末（2014年前後）由人口淨移入轉為淨遷出，柯文哲、蔣萬安兩任任內淨遷出持續擴大，2016–2025十年間累計流失約27.1萬人（約一個大安區）。這個趨勢和「施政滿意度」調查經常呈現不同方向——市長滿意度可以逆勢上升，但市民仍持續搬離，兩者背後反映的可能是不同層次的「宜居」定義（短期生活便利 vs. 長期居住成本）。
8. **國際參與有兩種路徑，效果不完全相同**：郝龍斌、柯文哲時期偏「賽事/文化型」（聽奧、花博、世大運、城市外交），能見度高、市民參與感強，但長期經濟外溢效益較難量化；蔣萬安時期偏「經濟投資型」（招商輝達），對城市長期稅收與產業結構影響可能更大，但市民日常感受度較低。這對「未來需要什麼樣市長」的討論也是一個值得深究的取捨。

## 8. 客觀數據補充：財政負債時間序列（1994–2023）

這是目前六任市長中，唯一有相對完整、跨任期可比對的**客觀量化指標**（其他如空氣品質、交通事故率等，詳見下節「待補充資料」）。單位：億元。

| 時間點 | 負債（億元） | 備註 |
|---|---|---|
| 1994（黃大洲卸任） | 約1,117 | 陳水扁任內減少約34億，故往前推算 |
| 1998（陳水扁卸任） | 1,083 | 4年任內減少約34億元 |
| 2002（馬英九連任時） | 1,633 | 馬第一任4年增加約550億元 |

| 2014（郝龍斌卸任／柯文哲上任） | 1,468.67 | 郝龍斌8年間負債由高點大幅下降 |
| 2018（柯文哲第一任末） | 898 | 首任4年還債570億元，創地方自治史上民選首長最高償債紀錄 |

| 2023（蔣萬安上任後） | 約824 | 上任後持續減債，人均負債由3.6萬元降至3.3萬元 |

**資料來源**：台北市政府財政局新聞稿、審計部歷年中央及地方政府決算審查報告、立法院質詢資料、聯合報/Yahoo新聞報導整理。

**解讀時的注意事項**：
- 「還債金額」常被政治人物用作政績宣傳（柯文哲自稱「還債570億」），但如上表所示，**只計首任、不計第二任**時數字會特別亮眼——這正是本研究一再強調的「政績量化容易各自解讀」的具體案例。
- 馬英九任內負債增加逾千億，但同期市政滿意度仍高達70–79%，顯示**市民對財政紀律的敏感度，遠低於對「有感建設」的敏感度**——這對「未來需要什麼樣市長」的討論很關鍵：財政穩健是否應該被賦予更高的公共討論權重？
- 黃大洲、陳水扁任期較短（4年），且早期資料揭露較少，負債數字精確度低於後續任期，圖表化時應標註「約」或使用區間表示。

## 10. 關於「模擬投票」功能的設計建議

由於你希望先聽建議，以下是三種可行方向，網站後續開發時可擇一或混合：

**方案A：政策傾向測驗（不提候選人名字）**
使用者對一系列政策題目（如「你認為社會住宅應該優先於財政紀律嗎？」）表態，系統依權重計算出使用者在「交通/財政/居住/環境」等維度的傾向分數，**最後才**比對到目前候選人的實際政見立場，給出「你與哪位候選人政見重疊度最高」的結果。
→ 優點：真正做到「跳開藍綠」，強迫使用者先想清楚政策立場，而非直接選人。

**方案B：政見/歷任市長政績對照表**
單純把柯文哲式的「政績量化」方法，用同一套客觀指標套用到現有候選人的公開政見上，做成類似本文件第0節的表格，讓使用者自行比較，不做總分排名（避免被質疑操縱結果）。

**方案C：兩者混合（建議）**
先做方案A的互動測驗（提高參與感、社群傳播性），測驗結果頁面附上方案B的政見對照表（提供理性佐證），並在頁尾明確聲明資料來源與方法論，維持中立可信度。
